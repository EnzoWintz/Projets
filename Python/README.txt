Ce projet s'illustrait dans le cadre d'un cours sur python avec comme but d'imprimer des photo sur une mini imprimante thermique

Script photomaton : 

Le script à exécuter en premier sera Print.py.


Quel script fais quoi : 

- Print.py = code contenant l'interface, fichier à lancer en 1er 

- main.py = code permettant la visualisation puis la prise de photo avec conversion de la photo en monochrome

- Config.sh = code permettant l'installation des packages nécessaire au fonctionnement du photomaton

- Logo.ico = icone de l'interface graphique

- SDV.png = logo utilisé lors de la visualisation de la prise de photo puis présent une fois la photo prise
